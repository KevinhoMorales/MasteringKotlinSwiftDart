# Notas del libro

Este repositorio complementa el contenido del libro:

**Dominando Kotlin, Swift y Dart**

Su objetivo es ofrecer un espacio práctico donde puedas explorar, ejecutar y experimentar con los ejemplos presentados a lo largo del libro.

Más que un simple repositorio, actúa como una extensión del contenido.

---

## Versiones de los lenguajes

Los ejemplos fueron desarrollados utilizando versiones modernas de cada lenguaje:

- Kotlin 1.9+
- Swift 5.9+
- Dart 3+

Es posible que, con el tiempo, aparezcan cambios menores en la sintaxis. Sin embargo, los conceptos fundamentales explicados en el libro se mantienen vigentes.

Cuando sea necesario, este repositorio se actualizará para reflejar dichas evoluciones.

---

## Entornos de desarrollo

Los ejemplos han sido desarrollados y probados utilizando:

- Android Studio  
- Xcode  
- Visual Studio Code (con Flutter)

Deberían funcionar correctamente en versiones recientes de estas herramientas.

---

## Actualizaciones

El desarrollo de software está en constante evolución. Por ello, este repositorio podrá incluir:

- mejoras en el código  
- nuevos ejemplos  
- ejercicios adicionales  

El objetivo es mantener el contenido útil y relevante en el tiempo.

---

## Enfoque del libro

Este libro se centra en:

- sintaxis de Kotlin, Swift y Dart  
- conceptos fundamentales de programación  
- comparación entre lenguajes modernos  

Estos fundamentos permiten adaptarse a nuevas herramientas y tecnologías con mayor facilidad.
