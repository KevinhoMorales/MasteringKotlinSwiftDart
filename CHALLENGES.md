# Dominando Kotlin, Swift y Dart

Este repositorio contiene los ejercicios prácticos del libro:

**Dominando Kotlin, Swift y Dart**

Su propósito es complementar el contenido teórico con ejemplos reales en tres lenguajes modernos utilizados en el desarrollo mobile:

- Kotlin  
- Swift  
- Dart  

Está diseñado como un entorno de práctica para experimentar, ejecutar código y consolidar conceptos.

---

## Estructura del repositorio

El repositorio (**MasteringKotlinSwiftDart**) está organizado como un monorepo con tres proyectos principales:

- MasteringKotlin → Ejemplos en Kotlin (Android / JVM)  
- MasteringSwift → Ejemplos en Swift (ecosistema Apple)  
- MasteringDart → Ejemplos en Dart (Flutter)  

Cada proyecto está organizado por capítulos.

Ejemplo de estructura:

- chapters/
  - chapter01/
  - chapter02/
  - chapter03/
  - ...

Cada capítulo contiene ejercicios con tres niveles:

- basic  
- intermediate  
- advanced  

---

## Cómo usar este repositorio

Para aprovechar el contenido:

1. Lee el capítulo correspondiente en el libro.  
2. Abre el proyecto del lenguaje que deseas practicar.  
3. Navega hasta la carpeta del capítulo.  
4. Ejecuta los ejemplos.  
5. Modifica el código y experimenta.  

La práctica es la base del aprendizaje.

---

## Cómo ejecutar los ejemplos

### Kotlin (Android Studio)

1. Abre el proyecto `MasteringKotlin`.  
2. Navega a la carpeta del capítulo correspondiente (por ejemplo: `chapters/chapter03/`).  
3. Abre el ejercicio correspondiente.  
4. Ejecuta el código usando Run o una función `main`.  

La salida se mostrará en la consola.

---

### Swift (Xcode)

1. Abre el proyecto `MasteringSwift`.  
2. Navega al capítulo correspondiente.  
3. Ejecuta la app o el ejemplo.  

La salida se mostrará en la consola de depuración.

---

### Dart (Flutter)

1. Abre el proyecto `MasteringDart`.  
2. Navega a la carpeta del capítulo (por ejemplo: `lib/chapters/chapter03/`).  
3. Ejecuta el archivo o la función `main`.  

La salida se mostrará en la terminal o consola.
